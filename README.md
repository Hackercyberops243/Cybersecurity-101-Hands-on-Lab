# 🔐 Cyber Security 101 – Hands-On Lab

This repository represents my hands-on learning journey through the TryHackMe Cyber Security 101 path.

Instead of only completing the course, I focused on applying key concepts using real-world tools and techniques.

---

##  Objective

Transition from:
 Understanding cybersecurity concepts  
 Applying them in practical scenarios  

---

##  Key Skills Developed

- Network scanning & enumeration (Nmap)
- Traffic analysis (Wireshark, tcpdump)
- Web testing (Burp Suite, SQLMap, Gobuster)
- Exploitation basics (Metasploit)
- Password attacks (Hydra, John the Ripper)
- Security monitoring & SIEM fundamentals

---

##  Practical Activities

###  Offensive Security
- Network scanning with `nmap`
- Directory brute-force using `gobuster`
- SQL injection testing with `sqlmap`
- Exploitation basics using `metasploit`
- Password brute-force with `hydra`

###  Networking & Traffic Analysis
- Packet inspection using `Wireshark` and `tcpdump`
- Understanding protocols and OSI layers

###  Web Security
- HTTP request/response analysis
- Burp Suite interception
- OWASP Top 10 awareness

###  Defensive Security
- SIEM fundamentals
- Log analysis
- Incident response basics
- Introduction to digital forensics

---

##  Key Takeaways

- Enumeration is one of the most critical phases
- Understanding systems is more important than tools
- Small misconfigurations can lead to major vulnerabilities
- Security requires both attacker and defender mindset
